# Legacy Architecture Archive

This directory contains the old layered planning system and related architecture notes from an earlier documentation model.

It is preserved because it still holds useful historical material:

- roadmap notes
- implementation plans
- benchmark writeups
- idea logs
- session handoff context

It is no longer the active structure for the repository.

Use instead:

- `docs/README.md` for the current docs index
- `docs/roadmap/current.md` for current direction
- `docs/reference/README.md` for practical repo entry points

Treat the `v1.0/` tree and `state/` metadata as legacy archive content, not as the source of current scope or workflow.
